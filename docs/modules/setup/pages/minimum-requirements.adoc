= Minimum Requirements

The requirements depend on the toolchain you choose to run Asciidoctor reveal.js.
In each case, all you need is the matching runtime, its package manager, and a compatible version of Asciidoctor.

[cols="1,2,2"]
|===
|Toolchain |Runtime |Asciidoctor

|Ruby / Bundler
|{url-ruby-download}[Ruby^] >= 2.7 and {url-bundler}[Bundler^]
|{url-asciidoctor}[Asciidoctor^] >= 2.0, < 3.0

|JavaScript (Node.js) / npm
|{url-nodejs-download}[Node.js^] >= 20 and npm >= 8
|{url-asciidoctorjs}[Asciidoctor.js^] >= 4.0, < 5.0
|===

For the full mapping between Asciidoctor reveal.js, reveal.js, and Asciidoctor.js versions, see the xref:compatibility-matrix.adoc[].
