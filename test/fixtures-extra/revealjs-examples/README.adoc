= Examples

Minimal AsciiDoc slide-decks to showcase reveal.js' features or test for regressions.

They double as the `revealjs-examples` family of the asciidoc-testkit fixture corpus (see `tasks/testkit.rake`), so some metadata (AsciiDoc comments `//`) is present.
A file here with no matching `test/expected-revealjs/revealjs-examples/<name>.html` is a pure showcase demo, not a regression test.

== Preview examples (ruby)

    $ bundle exec rake examples:convert

== Preview examples (javascript)

    $ bundle exec rake build:converter:opal
    $ npm run examples

Make sure you have a link between `test/fixtures-extra/revealjs-examples/node_modules/` and `node_modules/` where reveal.js is installed.
